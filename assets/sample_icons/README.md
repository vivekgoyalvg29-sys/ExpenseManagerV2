# Sample icon set for default categories and accounts

Handcrafted SVG icon set (128×128) for the built-in initialization data.

## Design notes
- Icons are now pictogram-based (no `<text>` nodes) for reliable Flutter SVG rendering.
- Each icon uses a soft background tile, colored circular badge, and white symbol.
- Palette is grouped by domain (income/expense/account) for faster visual recognition.
